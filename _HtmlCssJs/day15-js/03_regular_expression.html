정규 표현식 (Regular Expression)
문자 내에서 특정 내용을 찾거나 대체, 발췌 하는데 사용

  • regExp 단어로 많이 사용
  • 특정 데이터를 탐색, 확인, 존재여부를 쉽게 파악하기 위한 규칙
  • 웹 분야에서는 password, 첨부파일 규칙 등에 사용
    자주 사용되지는 않음
  • 데이터분석과 관련되서는 꼭 알고 있어야 함
  • 다른 언어에서도 거의 비슷한 형식으로 활용
  • /ragEap/i : (/ /) > 시작기호, 종료기호
    regExp : 패턴,i : flag(옵션)
  • ('str').match(/regExp/flag) : str 단어에서 매칭되는 항목들을 배열로 리턴
  • ('str').replace(/regExp/flag, '바꿀문자') : 정규표현식에 매칭되는 항목을 바꿀 문자로 변경
  • ('str').split(/regExp/) :
    정규표현식에 매칭되는 항목을 쪼개 배열로 리턴
  • (//regExp).test('str') : 문자열이 정규표현식 규칙에 맞는지 체크
    맞으면 true / 틀리면 false로 리턴
  • 정규표현식 플래그
  • i : 대소문자 구분 x / g : 전역검색 / m : 줄바꿈 포함
  • a-zA-Z : 영어 알파벳 / ㄱ-ㅎ가-힣 : 한글 / 0-9 : 숫자
  • . : 모든 문자열(숫자, 한자, 영어, 특수문자, 공백 포함) / 줄바꿈 x
  • \d : 숫자 \D : 숫자가 아닌것 (not에 의미)
    \s : 공백 \S : 공백 아닌것 \w : 영숫자 \W : NOT \특수기호 : \! , \@ ...
  • 검색하는 패던 : | or (a | b)
  • [] : 괄호안에 문자들 중 하나 / or의 묶음 문자들 중 하나
    /abc/ > 'abc'를 포함하는
    /[abc]/ > 'a','b','c' 중 하나를 포함하는
    /[a-z]/ > a-z 까지 중 하나를 포함하는
  • [^문자] : 괄호안의 문제를 제외한 것
    [^abcd] : a,b,c,d 제외 하고 나머지
  • ^문자열$ : ^(시작 표시) / $(끝 표시)
